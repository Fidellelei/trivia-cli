
PROJECT TITLE
West African Prose Literature Trivia CLI Game

PROJECT DESCRIPTION
This project is a Command-Line Interface (CLI) trivia game developed using JavaScript. The game allows users to answer trivia questions based on West African prose literature. Players receive feedback after each answer, complete the quiz within a time limit, and view their final score when the game ends.

The project demonstrates the use of foundational JavaScript concepts including arrays, objects, functions, loops, user input handling, and asynchronous timers.

OBJECTIVES MET
This project satisfies the required lab instructions by allowing users to:

* Start a quiz game
* View quiz questions sequentially
* Select answers to questions
* Receive immediate feedback on answers
* Receive feedback when the game is over
* Play within a timed session

JS FEATURES USED
The project uses the following JavaScript functionality:

* Functions
* Variables
* Arrays
* Objects
* Loops
* Array iteration method (`map()`)
* Asynchronous JavaScript timers (`setInterval()`)
* CLI user input with `readline`
